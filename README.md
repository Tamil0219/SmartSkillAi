# SmartSkill-AI 🚀
### Intelligent Learning & Adaptive Evaluation Ecosystem

**SmartSkill-AI** is a premium, full-stack educational platform built for high-performance digital learning. It features a clean, separated architecture for seamless scalability and high-quality user experiences.

---

## 💎 Project Architecture

The system is organized into two distinct, high-performance layers:

```bash
SmartSkill-AI/
├── frontend/  # React 19 + Vite + Tailwind UI Layer
└── backend/   # Node.js + Express + MongoDB API Layer
```

---

## 🛠️ Technology Stack

### Frontend (Ultra-Modern UI)
- **Framework**: React.js 19 (Vite)
- **Styling**: Vanilla CSS & Tailwind for premium aesthetics
- **Animations**: Framer Motion inspired micro-interactions
- **Icons**: Lucide React for crisp, scalable visuals
- **Navigation**: React Router 7 for fluid routing

### Backend (Robust API Infrastructure)
- **Runtime**: Node.js & Express.js
- **Database**: MongoDB (Mongoose Schema Architecture)
- **Security**: JWT Authentication & Bcrypt hashing
- **Access Control**: Role-Based Access Control (RBAC)

---

## ⚡ Core Features & Roles

- **👑 Administrator**: Full-spectrum control, user management, and system auditing.
- **🎓 Mentor**: Cohort tracking, course authoring, and advanced student analytics.
- **💡 Student**: Dynamic learning paths, AI-powered adaptive exams, and performance tracking.

---

## 🚀 Quick Start Guide

### 1. Backend Setup
```bash
cd backend
npm install
# Ensure MongoDB is running
npm run dev # Runs at http://localhost:5000
```

### 2. Admin Seeding
```bash
cd backend
node seedAdmin.js # Default: admin@smarteval.ai / admin123
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev # Runs at http://localhost:3000
```
